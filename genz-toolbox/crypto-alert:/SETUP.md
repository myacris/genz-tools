# Crypto Alert Bot Setup

1. Install Python 3.8+ from python.org
2. Open terminal in this folder
3. Run: pip install -r requirements.txt
4. Edit crypto_bot.py and add your Telegram Token and Chat ID
5. Adjust price thresholds (BTC_HIGH, BTC_LOW, etc.) to your preferences
6. Run: python crypto_bot.py

To run 24/7: Use a VPS or run on your local machine
